# React + Vite + Tailwind + Framer Motion Template

This is a modern web application template initialized for high-performance and beautiful animations.

## Tech Stack
- **React 19**
- **Vite**
- **Tailwind CSS v4**
- **Framer Motion**

## Project Structure Setup
The project comes pre-configured with the following structure to ensure scalable development:

```text
src/
├── assets/         # Static assets like images and global icons
│   └── images/     
├── components/     # Reusable UI components (Buttons, Cards, Inputs)
├── context/        # React Context API for global state management
├── hooks/          # Custom React hooks (e.g., useFetch, useAuth)
├── pages/          # Application views/pages (Home, About, Dashboard)
├── styles/         # Additional complex CSS/SCSS (if needed beyond Tailwind)
├── utils/          # Helper functions, formatters, and constants
├── App.jsx         # Main application root component
├── index.css       # Tailwind entry and global CSS variables
└── main.jsx        # React DOM entry point
```

## Getting Started

1. **Install Dependencies**
   ```bash
   npm install
   ```

2. **Start Development Server**
   ```bash
   npm run dev
   ```

3. **Build for Production**
   ```bash
   npm run build
   ```
